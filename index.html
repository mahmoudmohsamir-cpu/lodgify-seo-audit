My SEO Methodology

How I Approach SEO — Research, Analysis & Integration

I always start with the competitive landscape before touching a single line of content. Every engagement follows a structured 3-step process: understand competitors → audit own site → integrate all signals. The entire workflow is grounded in manual research first, then validated and scaled with tools and AI automation.

1
Competitor Analysis — Manual First, Then Tools

Before looking at our own site, I spend time manually browsing the top 5 competitors' highest-ranking pages — how they structure their H1/H2 hierarchy, what schema markup they implement, how their FAQ sections are formatted, what their content depth is per target keyword, and how they handle internal linking. I then layer on dedicated SEO tools (Ahrefs and Semrush) to pull keyword gap data: what keywords competitors rank for that we don't, where their backlinks come from, and what SERP features they're winning. Manual review shows me strategy; the tools show me scale.

Manual SERP Review Keyword Gap (Ahrefs / Semrush) Competitor Backlink Audit SERP Feature Mapping
2
Own Site Deep-Dive — 4-Source Analysis

I mirror the same review on the client's site using four data sources simultaneously. Google Search Console shows me which keywords we already rank for (even partially), CTR vs. impressions gaps, and position trends — this is my first keyword research source because it's ground truth, not estimates. Google Ads search terms (when active) reveal high-intent, revenue-validated keywords the audience already uses to find us — an often-ignored goldmine for organic keyword targeting. Ahrefs / Semrush site audit surfaces technical issues, orphaned pages, and keyword ranking distributions. And a manual crawl of top pages validates what the tools can't see — actual UX, content quality, schema implementation, and internal link anchor text.

Google Search Console Google Ads Search Terms Ahrefs / Semrush Audit Manual Page Crawl
3
Integration — All 4 Sources, Automated with Claude

The real insights emerge at the intersection of all four signals: Google Analytics (behavior: bounce rate, session time, scroll depth, conversion paths), Google Search Console (impressions, CTR, position by keyword), Ahrefs / Semrush (authority, backlink quality, keyword gap with competitors), and Claude AI for automation and pattern recognition at scale. When a page shows declining impressions in Search Console, low time-on-page in Analytics, a competitor keyword gap in Ahrefs / Semrush, and no schema from the audit — that's a clear, quantified, prioritized fix. I use Claude to automate this cross-referencing: flagging priority pages, generating schema markup drafts, mapping content gaps to search intent, and producing priority matrices — work that would take days manually is done in hours.

🔑 My Keyword Research Sources — All Four, Every Time

I never rely on a single keyword source. I always combine: (1) Google Search Console — what we already rank for (the safest expansion territory); (2) Google Ads search terms — actual user queries that converted or engaged, directly mapped to organic opportunities; (3) Ahrefs / Semrush competitor keyword database — what competitors rank for that we don't (the gap), and what both of us rank for but they rank higher (the weak spots); (4) Manual search exploration — typing queries into Google myself to see People Also Ask boxes, AI Overviews, featured snippets, and related searches that tools often miss. These four sources together create a complete, validated keyword universe — not just a list, but a prioritized, intent-mapped content plan.

My Strategy Across the 3 SEO Layers
📝
On-Site SEO

Full content audit first: H1/H2 structure, title-to-H1 alignment, keyword density, internal link flow, schema gaps. I prioritize pages by traffic potential × current position — pages ranking 6–15 get attention first (fastest wins). Every high-value page gets FAQPage + Article schema, an answer-first intro rewrite for GEO, and a content depth review against top-3 competitors for that keyword. Thin content always gets expanded before new content is written.

🔗
Off-Site SEO

Off-site is a content and relationship play, not a link-buying operation. Strategy: (1) original data reports for editorial links from industry media — highest-trust links; (2) expert responses to journalist queries for high-authority publications; (3) community partnerships with forums, host groups, and YouTube creators for brand mentions that AI models index; (4) digital PR around product milestones. 10 editorial links from high-authority domains outperform 500 directory listings every time.

⚙️
Technical SEO

Technical is always week-one priority — content work is wasted if crawlers can't access it. Full site audit using Ahrefs + manual checks: crawl errors and broken links first, then pagination (JS "Load More" → HTML), then Core Web Vitals (especially mobile LCP).
